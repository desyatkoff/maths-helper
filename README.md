# MathsHelper

## Description
MathsHelper is a CLI tool for doing mathematics things such as finding area of square etc...

## Features
MathsHelper **can find** ...:
1. Area of **square**;
2. Area of **circle**;
3. Perimeter of **square**;
4. **Hypotenuse**;
5. **Circle** length.

*That's all yet, but I will update this project soon. :)*
